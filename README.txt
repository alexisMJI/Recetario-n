Proyecto en el cual puedes crear un usuario y realizar el Login.
En donde te va a permitir crear modificar y borrar tus propias recetas.
Trabaje desde el lado del Front-end utilizando React, Axios, Tailwind realizando las peticiones a dos APIs una para las recetas y la otra para los usuarios
A la hora de realizar el Login y crear  las recetas se utiliza JWT para autenticar que las peticiones son validas.
Las rutas s encuentras protegidas de tal modo que si no te logueaste correctamente no podras acceder al panel de las recetas aunque ingreses la url
